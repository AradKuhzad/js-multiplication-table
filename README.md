# Multiplication Table Generator

A simple JavaScript project that creates a multiplication table using nested loops.

## Features

* User input for rows and columns
* Generates a multiplication table dynamically
* Built with HTML, CSS, and JavaScript
* Great practice for nested loops and DOM manipulation

## Technologies

* HTML5
* CSS3
* JavaScript

## What I Learned

* Working with nested loops
* Taking user input
* Manipulating the DOM
* Creating dynamic HTML content

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter the desired number of rows and columns.
4. Generate the multiplication table.
